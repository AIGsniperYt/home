Pointing Navigation Simulator

Simulate how a robot would interpret pointing gestures for navigation. Create walls to represent obstacles, then right-click to set origin and target points.

How to Use:

  • Left-click to toggle walls (obstacles)
  • Right-click first on the origin point (your hand position)
  • Right-click again on the target point (where you're pointing)
  • An arrow will be drawn representing the pointing direction and magnitude
  • Use the control panel to reset or run simulations
